Project summary

SDAT is an acronym that stands for "Shipboard Dimensional Analysis Tool". SDAT is a digital twin tool that combines point cloud scans of large military or industrial ships with 3D models of vehicles and equipment that need to be stored inside them. By combining these models, users of this technology are able to pre-visualize the packing of cargo that needs to happen on the ship. The SDAT tool provides pathfinding analysis, can identify potential collisions between the cargo and the ship, and will auto generate PDF reports for the user.

My role

My role on this project was as the lead UX/UI designer. I guided the development of the interface, created a design system, mocked up user flows, and proposed/prototyped features for the final product.


I also acted as the SDAT team's UI integrator/front-end developer. I took the UI designs mocked up in Adobe XD and built them in our Unreal Engine 4 software prototype. I used unreal's visual scripting language to create our hover effects, sliders, radio buttons, and other UI widgets as well as a tool tip system
